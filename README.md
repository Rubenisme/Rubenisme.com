# Rubenisme.com

My personal website — a mix of professional info and a bit of fun.

Live at **[rubenisme.com](https://rubenisme.com)**.

## What's on it

A single-page site with a short intro about who I am, what I work on, and how to get in touch. 
It also has a few personal touches to keep things from feeling too corporate — including a custom boat cursor that leaves a water wake as you move your mouse around.

## Built with

- [Astro](https://astro.build) — static site framework
- Vanilla JS for the interactive bits
- Docker + Traefik for self-hosted deployment
